3. what are the four pillars of OOPs? 
    üîπ 1. Encapsulation ‚Äì "Data hiding"
            It means wrapping data and code together inside a class.
            We hide internal details and only expose necessary parts.
            Achieved using private variables and public getter/setter methods.
        
            eg. Car class with private speed, accessed via methods.
    
    üîπ 2. Abstraction ‚Äì "Hiding complexity"
            Shows only important things, hides complex internal logic.
            Done using abstract classes or interfaces.
    
    üîπ 3. Inheritance ‚Äì "Reusing code"
            One class (child) can reuse code from another class (parent).
            Helps reduce code duplication.
    
    üîπ 4. Polymorphism ‚Äì "One name, many forms"
            Same method behaves differently depending on the object.
            Two types: Compile-time (method overloading) and Run-time (method overriding).

4. How do call one constructor from another?
        we can achieve this using constructor Chaining.
        Constrcutor chaining is the process of calling one constructor from another within the same class or from the base class.
        Purpose : To reuse code and avoid duplication when initialising an object.

        Two types:
            Within the same class : this() keyword.
            from base/parent class : super() keyword.

        class Animal {
            Animal(String type) {
                System.out.println("Animal constructor called: " + type);
            }
        }

        class Dog extends Animal {
            Dog() {
                this("Tommy");  // Calls the second constructor of Dog
                System.out.println("Default Dog constructor"); //3
            }

            Dog(String name) {
                super("Dog");  // Calls parent class constructor //1
                System.out.println("Dog constructor with name: " + name); //2
            }
        }

        public class Main {
            public static void main(String[] args) {
                Dog d = new Dog();
            }
        }

        OUTPUT:
            Animal constructor called: Dog  
            Dog constructor with name: Tommy  
            Default Dog constructor
